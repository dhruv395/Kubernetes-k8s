##Why we need Kubernetes Services
- to have a permanent IP address
- to make various components/pods connect and communicate
- to make applications exposed to outside world

##What is a Kubernetes Service
- is a grouping of pods that are running on the cluster.

  features are:
- load balancing
- service discovery between apps
- zero downtime app deployment


Types of Kubernetes Services
- ClusterIP
- NodePort
- Load balancer
